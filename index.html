CSS Cheatsheet

Block Element Reference

Tag Name Description
<p> Paragraph A simple Paragraph
<blockquote> Block Quotation Element. Indicates that the enclosed text is and extended quotation.
<ul> Unordered List Generates and unordered list with bullet points
<ol> Ordered List Generates and ordered numbered list

Inline Element Refrence

Tag Name Description
<a> A link Links to another source
<img> An image Sources and image
<em> Italic Text Changes text to italic
<b> Bold Text Changes text to bold
